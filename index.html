JamRadio

My final project at Launch. Using SoundCloud API I built an application similar to Pandora with a few more features

JamRadio has 3 different playlist. The first one is a list of radio stations where depending on the genre or the name of the artist the app generates a playlist for the user. The second one is a list of the user's favorite songs , he can add songs individually or with the like button adding the song that the radio is playing. The last one is a list of groups, the user can create a group or be just a guest. Then every member of the group can add songs to the same playlist, making share music easier.
